i18n-service-demo works!

How to use i18n:

  • How to DI service:

    1. inject the service in corresponding module providers:

    please see:

    ~/televic-i18n-service/src/app/i18n-service-demo/i18n-service-demo.module.ts

    2. inject the service in constructor

    please see:

    ~/televic-i18n-service/src/app/i18n-service-demo/i18n-service-demo.component.ts

    then we can use the function in service to emit api call

How to use i18n:

  • which language do we use:

    1.set language:

    the language get by "navigator.language" as default, once you need to select one language, you must add language options to:

    export enum Language{{'{'}} zh_CN = 1, en_US, {{'}'}} const BCP47Mapping = {{'{'}} en_US: 'en-US', zh_CN: 'zh-CN', {{'}'}};

    and then have frontEnd selector or backend api to get "language" in BCP47 format and set it into localstorage:

    {{"localstorage.setItem('tec_language', language)"}}

  • How to use, please go to see:

    in your corresponding module: '~/televic-i18n-service/src/app/i18n-service-demo/i18n-service-demo.component.ts'

    in your corresponding routing module: '~/televic-i18n-service/src/app/i18n-service-demo/i18n-service-demo-routing.module.ts'

    in sibling folder of: ~/televic-i18n-service/src/i18n, add the namespace and key values with indent as format:

    i18nServiceDemo: name: 姓名 sex: 性别 age: 年龄

    then run script in terminal under project root folder: npm run i18n

  • how to translate:

    translation in typescript: use translate service

    take a look at the valiable of "translatedNameLabel":

    ~/televic-i18n-service/src/app/i18n-service-demo/i18n-service-demo.component.ts

    translation in template: add translate pip

    example case is in below code of template.

{{'i18nServiceDemo.name' | translate}}
{{'i18nServiceDemo.age' | translate}}
{{'i18nServiceDemo.sex' | translate}}
